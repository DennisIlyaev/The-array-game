Instructions

  1. Set an array, choose how many items will be in the array and afterwards insert items (NUMERIC ONLY) in each index of the array.
  2. After array has been declared, click on the header "Array#1/2/3" on the left side, will be highlighted in blue.
  3. When your array is declared + you chose what array to work with (Highlighted) choose an action to work with. Results will either be written below with green color or will modify the actual array.
  4. To reset press F5 or just re-Set array#1/2/3. You can work with different arrays at the same time, and also able to combine actions (Use common sense)
